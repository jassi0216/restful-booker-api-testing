# Restful Booker API Testing Project

This project demonstrates API testing using Postman on the Restful Booker API.

Tools Used:
- Postman
- REST APIs
- JSON

Testing Coverage:
- GET Requests
- POST Requests
- PUT Requests
- PATCH Requests
- DELETE Requests

Features Implemented:
- API chaining
- Environment variables
- Global variables
- Pre-request scripts
- Post-request scripts
- CRUD operations validation
